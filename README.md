# Weather App
## Requirements
1. Create a browser or native-app-based application to serve as a basic weather app.

2. Search Screen

3. Allow customers to enter a US city

4. Call the openweathermap.org API and display the information you think a user would be interested in seeing. Be sure to has the app download and display a weather icon.

5. Have image cache if needed

6. Auto-load the last city searched upon app launch.

7. Ask the User for location access, If the User gives permission to access the location, then retrieve weather data by default

## Language & Tools used
* Coding Language - Kotlin
* Architecture Pattern - MVVM
* Network library - Retrofit
* Concurrency - Coroutines
* UI - Jetpack Compose
* Dependency Injection - Dagger Hilt
* Navigation - Jetpack Navigation
* Image Loading library - Coil
